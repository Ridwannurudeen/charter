# Charter module registry

This is the human-reviewed marketplace inventory. It is not an on-chain allowlist or security approval. Read the
[listing policy and security boundaries](README.md) before using any entry.

The Sepolia addresses below come only from the repository's deployment records. The frozen Phase 0 records do not assert
that current source compiles to the same runtime bytecode. In particular, the active deployed `DividendDistributor`
predates the Phase 1 `ledgerVersion` remediation documented in the
[security self-review](../SECURITY-REVIEW.md#c-02-distribution-record-ledger-drift). The Phase 2 wrapper and resolution
were deployed from the current source and source-verified on Etherscan; this does not prove their unexecuted Sepolia
vote flow or an external integration.

First-party entries are reference inventory and do not count toward the Phase 2 external-module gate. A pending or not
applicable conformance status is not equivalent to a pass.
